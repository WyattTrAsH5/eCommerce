# Wyatt Reeve's eCommerce
This is a website to learn the fundamentals of ASP.NET Core MVC (Model View Controller).

## Features
- Dark themed UI styled with Bootstrap 5
- CRUD (Create, Read, Update, Delete) operations for products with Entity Framework Core
- User Login and Registration with ASP.NET Core Identity
- MVC (Model-View-Controller) architecture throughout

## Technologies/Frameworks Used
- ASP.NET Core MVC with C#
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server (LocalDB)
- Bootstrap 5
- HTML/CSS/JS
